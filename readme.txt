Let user can pay how much they like by setting one or more product with this plugin

== Description ==

Create your own product with free price and use this plugin to take donations. Works with all payment method.

* Select only one product or many products to set with free price
* Set product like donation product
* See the total donation ammount

*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at report@nsthemes.com.*

== Installation ==

1. Upload the entire 'ns-free-price-and-donation-for-woocommerce' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Settings' -> 'NS Free Price and Donation for WooCommerce'


== Frequently Asked Questions ==

= Is required WooCommerce to use this plugin? =

Yes WooCommerce is required to use this plugin.

= This plugin hide cart page and checkout page? =

Yes, if you enable option cart page and checkout page doesn't exist and if you go to /cart/ or /checkout/ you are redirect to home
